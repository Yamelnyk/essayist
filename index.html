How It Works

  • Choose Citation Style

    Essayist currently supports APA, MLA, Chicago/ Turabian & Harvard. Stay tuned for more citation styles!

  • Add References

    Use Essayist’s Reference Manager & built-in Google Scholar support to easily add references at any time.

  • Write & Cite

    Let Essayist handle the formatting and begin writing your essay stress-free! Insert citations with the click of a button.

Automatic
Formatting

Let Essayist handle the formatting so you can focus on the content of your essay. From page set-up to citations, footnotes & references, Essayist takes care of everything.

Reference
Manager

Add and manage references using Essayist’s built-in reference manager. Create collections and easily add sources and Essayist will generate a perfectly formatted reference list.
